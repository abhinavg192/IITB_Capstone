# AI Social Media Agent
IIT Bombay e-PG Diploma in AI & DS — Capstone 2026

An AI-powered social media content generation system that generates 
platform-specific posts with brand voice alignment, trending topic 
injection, and ML-based engagement scoring.

## Repo Structure
- `notebooks/` — each member's Colab notebooks (work in progress)
- `modules/` — finalized Python modules ready for integration
- `data/brand_guidelines/` — sample brand documents
- `models/` — trained model .pkl files
- `app/` — Streamlit application

## How to Use
